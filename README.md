# BepInEx-Installer
Fully Installs BepInEx to your Gorilla Tag Directory!

# HOW TO USE

1 - Download The BepInEx.bat

2 - Right click the file and press open

3 - Click any key to continue once finished

4 - Run your game and wait for plugins to show up in your BepInEx folder

5 - Add mod and rerun game
